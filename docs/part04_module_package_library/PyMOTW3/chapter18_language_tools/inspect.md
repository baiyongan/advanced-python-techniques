# inspect 模块

!!! note
    inspect 模块提供了对活动对象及其源代码进行内省的功能。

    inspect 模块提供了学习活动对象的功能，包括模块、类、实例、函数和方法。 此模块中的函数可用于检索函数的原始源代码，查看堆栈上方法的参数，并提取对生成源代码的库文档有用的信息。
