## 学习路径

- 首先，任何技术的学习不要脱离官方文档。SWIG 网站上提供了难以置信的详尽文档，通过文档掌握 SWIG 的用法，显然是最好的一个途径。

- 其次，要深入 SWIG，需要对 C++ 有一个较为全面的掌握。对于高性能计算来说，C++ 总是绕不开的一个主题，特别是对内存管理、指针和虚函数的应用，需要实际上手编写 C++ 代码后，才能逐渐掌握。
